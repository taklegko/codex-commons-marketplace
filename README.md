# Codex Commons Marketplace

Public Codex marketplace mirror for Codex Commons.

Install it with:

```bash
codex plugin marketplace add taklegko/codex-commons-marketplace
```

The source of truth is generated by Codex Commons and mirrored into `.agents/plugins/marketplace.json`.
